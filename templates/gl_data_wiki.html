Database

Le menu "Database" présente les fonctionnalités suivantes:

  • Games: Fourni un liste aléatoire d'une vingtaine de jeux de la base
  • Add Game: Permet l'ajout d'un jeu dans la base de données via un formulaire
  • Refresh Database: Permet le rafraîchissement de la base de données via les API des différents services
  • Roulette: Propose un jeu aléatoire de la base qui n'est pas topé comme "fini"
